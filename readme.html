Read Me

  • App is split into block elements that can be position or hid.
  • The top level block element is #new_city_player, which is a fixed width (1920 x 1080).
  • Other Player components are all siblings. Elements have Id values and have child div elements with class component_content. Components that are clickable have child div elements that have the class click_control.
    1. #top_nav - Not implemented.
    2. #company_logo - A space for a company logo.
    3. #current_time - The current time. May need localization.
    4. #message - The Message. The primary content.
    5. #duration - The countdown until Message update.
