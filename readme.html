Read Me

  • The top level block element is #new_city_player, which is a fixed width (1920 x 1080).
  • #new_city_player is split into block elements that can be position or hid.
    1. Components have Id values and have child div elements with class nc_component. Components may have child elements called Lables with the class nc_label. Labels have the additional class of pre or post, dependeing on wheather the Label appears before or after the nc_component tag.
    2. Controls have the class nc_control, and a child element that has the class click_control or select_control. Controls have children with the classes control_icon, control_text or both. Controls can also have Labels.
    3. ComponentGroups can group Components. They have the class nc_component_group. Specified below.
    4. ControlGroups can group Controls. They have the class nc_control_group. Specified below.
    5. Tickers have the class nc_ticker. Tickers have children with class .ticker_inner which has children called .ticker_item.
